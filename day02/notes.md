# Day 2 Exercises

## Question 1: What is a CPU?

A CPU (Central Processing Unit) is the brain of a computer. It executes instructions, performs calculations, and manages system operations.

---

## Question 2: What is RAM?

RAM (Random Access Memory) is temporary memory used by the computer to store data that is actively being used by applications and the operating system.

---

## Question 3: What is an Operating System?

An operating system is software that manages hardware resources and provides services for applications to run.

Examples:
- Windows
- Linux
- macOS

---

## Question 4: What is Virtualization?

Virtualization is the process of creating virtual versions of computers, servers, or operating systems that run on a physical machine.

---

## Question 5: What is a Virtual Machine?

A Virtual Machine (VM) is a software-based computer that runs inside another computer.

Examples:
- Ubuntu VM
- Kali Linux VM
- Windows VM

---

## Question 6: Why do Cybersecurity Professionals use Virtual Machines?

Cybersecurity professionals use virtual machines to:

- Create testing environments
- Practice safely
- Analyze malware
- Build cybersecurity labs
- Learn different operating systems

---

## Summary

Today I learned the fundamentals of computer hardware, operating systems, virtualization, and virtual machines, which are essential concepts for building a cybersecurity lab.
